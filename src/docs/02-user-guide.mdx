# DocsKit user guide

In this user guide you'll learn about: 

- The [Markdown markup language](/user-guide/markdown/) that you can use to add formatting elements to your documentation pages
- The [custom components](/user-guide/components/) available for use in your MDX documents, enabling you to add additional styling and functionalities
- The [navigation system](/user-guide/navigation) DocsKit uses to generate the website structure, main menu, and sidebar navigation
- The [theming layer](/user-guide/theming) of Docskit
